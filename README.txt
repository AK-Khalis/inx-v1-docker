To download and use the docker version of Inspextor application, do the following
1. Clone this repository to your target environment
2. run sh downloadApplication.sh to download the docker images to your target environment.
3. run sh startApplication.sh to start docker containers for the Inspextor app.
4. run sh stopApplication.sh to stop the docker containers.

Once docker containers are started, open the following URL in your browser to access the application
http://<IP of target Environment>/inxs/
